Moje Umiejętności

HTML 5

Osobiście myślę, że dobrze opanowałem język znaczników HTML i szybko odczytuje obcy kod.

JavaScript

Napisałem w tym języku tysiące lini kodu, na strony jak i do silnika gier Unity, więc myślę, że radzę sobie w nim bardzo dobrze.

CSS

Może nie znam wszystkich selektorów jak i funkcji wbudowanych w CSS3 lecz sądzę, że ten język mam przyswojony w podobnym stopniu co HTML.

PHP

Język serwerowy PHP mam opanowany w bardzo dobrym stopniu, niestety nie jestem przysfojony z żadnym frameworkiem dla tego języka.

jQuery

Z jQuery zwykłem pracować zawsze, gdy była taka możliwość, ułatwiając sobie tym pracę w JavaScript.

MySQL

Wszystkie operacje na bazach danych wykonywałem za pomocą MySQL. Znam podstawy języka SQL.

Bootstrap

W większości moich projektów podstawowym zestawem przygotowanych wcześniej klas i funkcji, co przełożyło się na moje obecne umiejętności.

Wordpress

Wordpress nie jest trudny w obsłudze jak i modyfikowaniu go za pomocą wszelakich pluginów. Cięższą pracą jest pisanie tych pluginów, co zresztą wiem, ponieważ sam próbowałem napisać szablon od podstaw.

Photoshop

Photoshop nie jest mi obcy, potrafię stworzyć jak i modyfikować różne prostsze grafiki. Lecz z tymi bardziej zaawansowanymi wyzwaniami nie dałbym sobie rady.

C#

C# i platforma .NET to język jak i platforma, od której zaczynałem swoje przygody z programowaniem. Niestety zaniedbałem moje umiejętności w tym języku, na rzecz innych języków wymienionych wyżej. Mam zaś zamiar powrócić do tego języka i bardziej rozwijać moje umiejętności w tym kierunku, jak i webowe i desktopowe.

Sass

Język Sass jest postprocesorem języka CSS który dobrze znam, lecz posiada wiele funkcjonalności których się dopiero uczę i wdrażam do moich projektów.

Git

Na systemie Git nie znam się zbyt dobrze, lecz potrafię z niego korzystać, pracować na gałęziach, łączyć i porównywać, wypychać swoje postępy w pracy.

Unity

Wielu programistów zaczynało swoje przygody od pisania prostych gier. Ja też tak zaczynałem. W Unity starałem się tworzyć rozmaite gry lub animacje. Moje gry nie ujrzały światła dziennego aż do dziś.

English

Moją znajomość języka angielskiego oceniam na stopień dobry, co umożliwia mi czytanie dokumentacji technicznej ze zrozumieniem.
